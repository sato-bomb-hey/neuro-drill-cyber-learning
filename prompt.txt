# MISSION: Data Injection & Button Activation

タイトル画面の UI 実装は完了しましたが、「DRILL START」ボタンが非活性（または反応しない）状態です。
以下の手順で、学習コンテンツを有効化してください。

1. **問題データの生成と投入**:
   - 各教科（国語・数学・理科・社会・英語）200問、計1000問の JSON データを生成せよ。
   - 作成したデータを Firestore の `questions` コレクションへ一括アップロードするスクリプト `scripts/seed-questions.ts` を作成し、実行せよ。
2. **ボタンのロジック確認**:
   - `DRILL START` ボタンの `disabled` 条件を確認せよ。
   - Firestore から問題データが正常に取得できている場合に、ボタンを活性化（クリック可能）にし、問題画面へ遷移するように修正せよ。
3. **データフェッチの実装**:
   - 選択された「学年・期・教科」に合致する問題を 10 問ランダムに取得するロジックを実装せよ。

まずは、問題データの生成から開始してください。
